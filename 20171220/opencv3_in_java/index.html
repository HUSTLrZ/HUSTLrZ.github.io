Opencv3 in java 学习笔记：一些应该写在前面的话

2017-12-20 | 2017-12-21 | opencv | 次

　　最近在做老师安排的图像识别的工作，而我之前是没有这方面的经验的，甚至说，连做一个完整项目的经验都没有。
　　早觉得该写些东西，写给自己看也好，分享给需要的人也好。把自己的工作记下来，日后若用的到，就翻出来看看，所以我把这些称为笔记。既然是笔记，那就只是我所学到的或看到的，是否是绝对的正确的知识，这一点我是保证不了的。你如果读到了，那看看就好，作个参考，自己多尝试，问题总会有解决的办法。
　　之前对图像识别这块了解甚少，这应该是属于计算机视觉领域的一小类，然而在去了解一些后，才知道原来这个领域已经如此成熟，已经有如此丰富的开源库，如果不是做行业创新而只是产品开发，那么拿来用就是。
　　关于opencv我不多做阐述，如果还未了解，建议先看一些这个：
　　OpenCV 中文教程
　　这是官方的系列教程，对于了解其中原理很有必要，而我是不会C++的，看看还好，要自己写就有点捉急了。尽管如此，老师给安排了我还是硬着头皮上了，在我了解到opencv有了java实现后，便松了一口气，至少还可以往下看。然而在之后的学习过程中又遇到了一个问题，尽管opencv有了java实现，但是程序员们似乎对用java写图像识别并不感兴趣，网上的资料少之又少，我想大概是考虑到图像识别的效率问题吧。其实opencv的java API底层是调用的native方法，虽说效率无法和C++相比较，但做做研究，效果也还差强人意。
　　我在程序的编写过程中遇到不少API的使用问题，走了不少弯路，现在把已经证实的使用方法记下来，别人用得到自然是好的，用不到，也是对自己工作的一些总结。
　　如此。
